Git

  • git init — инициализирует репозиторий Git в выбранной папке.
  • git clone — скачивает удалённый репозиторий и создаёт его локальную копию.
  • git status — выводит текущее состояние рабочей директории и индекса (какие файлы изменены, добавлены, неотслеживаемые и т. д.).
  • git branch — отображает перечень всех локальных веток проекта.
  • git checkout — осуществляет переход на указанную ветку.
